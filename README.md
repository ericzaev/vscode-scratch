# Scratch Files for VS Code

JetBrains-style scratch files for VS Code. Access and manage global scratch files from any project via a dedicated sidebar.

## Features

- **Global Scratches:** Scratch files are stored in the VS Code global storage directory, making them accessible across all workspaces and projects.
- **Sidebar Integration:** A new icon in the Activity Bar provides a dedicated view for all your scratch files.
- **Create New Scratch:** Quickly create a new scratch file with a specific language/extension.
- **Manage Scratches:** Rename, delete, and refresh scratches directly from the sidebar.
- **Automatic Refresh:** The sidebar automatically updates when files are created, renamed, or deleted.

## Usage

1. Click on the **Scratch Files** icon in the Activity Bar.
2. Click the **+** (Create New Scratch) button in the view title.
3. Select a language for your scratch file.
4. Enter a name (or use the default timestamp-based name).
5. The file will open automatically and be listed in the sidebar.

## Commands

- `Scratch: Create New Scratch`: Creates a new global scratch file.
- `Scratch: Refresh Scratches`: Manually refreshes the scratch file list.
- `Scratch: Delete Scratch`: Deletes the selected scratch file.
- `Scratch: Rename Scratch`: Renames the selected scratch file.
